Informatiile privind vremea, sunt obtinute cu ajutorul unui API de pe openweathermap

Pe prima pagina se poate observa vremea la momentul actul in orasul specificat

Pentru a vedea vremea pe mai multe zile pentru orasul specificat este nevoie sa faceti un cont

Contul se face rapid, si fara verificare, un sistem de login simplu scris in PHP

Aplicatia este stilizata folosint framework-ul Tailwindcss
